What is HTML?

HTML or Hyper Text Mark up Language are the heart of the web,
the web is made of of html documents that can do a number of things

HTML is made up of:

  1. Text Content - what you see
  2. Mark up - what it looks like
  3. References to other documents - e.g. images & vidoe
  4. Links to other pages

How do we make text look different in the browser?

We make text look different by using markup. Markup is made up of tags e.g. content.

The tag itself is made up of a less than and more then arrow with
the tag name inbetweeen this is called an opening tag. A closing
tag is made up of a less then sign followed by a forward slash,
the tag name and a more the sign this is called a closing tag. The content
lies content that you wish to appear in the browser is housed between
the opening and closing tag.
