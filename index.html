Lesson 1: The Web & HTML

The web explained

The World Wide Web was invented in the early 1990's by Tim Berners-Lee a British computer scientist. The web is a system of internet servers / computers that support a specially formatted document and through this can communicate with each other. These documents are formatted in a markup language called HTML (HyperText Markup Language) that supports links to other documents, as well as graphics, audio, and video files. So for example when you search in your browser for a website say www.google.ie what happens is your computer mkaes a HTTP request to the server. The server then finds the file and send this back to your browser which interprets the page and displays it in your browser window.

HTML, What is it

HTML stands for Hyper Text Mark up Language and is the heart of the web. It is a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.

Tags & Elements

HTML code is made up of characters that live inside angled brackets, these are called Elements. Elements are made up of of two tags an opening and a closing tag.

Block Elements & inline

The 'br' tag is an inline tag a 'p' tag is a block tag. What this means is that the br tag for ends a line. While the 'p' tag works differently the 'p' tag actually makes an invisible box, this can have height and width where as inline elements are just text. inline elements are: 'b' 'em' and even 'img' tags. A block element Would be the 'p' tag.

Lesson 2: Building a HTML Page

Developer Tools

Dev tools let developers look under the hood of what the browser is doing whenever the browser renders a page. You can make changes to the html within dev tools but these HTML changes made in Dev Tools will not get saved to our actual files.

Dev tools can be opened by clicking on the Hamburger icon that appears on our upper right hand side, then click "More Tools" and then click "Developer Tools".

HTML 'Tree Structures'

Refer to the hierarchal nature of HTML. Document Object Model - a cross-platform and language-independent convention for representing and interacting with objects in HTML (and other markup languages). The nodes of every document are organized in a tree structure, called the DOM tree.

Boxes

Everything on a webpage is a box. Thinking of a web page in terms of boxes makes your life as a web developer a lot easier. By thinking of things in terms of boxes we can quickly rearrange things if you decide to give users a different experience.

Text Editors

A text editor is a type of computer program that edits plain text. Such programs are sometimes known as "notepad" software.Text editors are provided with operating systems and software development packages, and can be used to change configuration files, documentation files and programming language source code.

An example of a text editors would be sublime or for exmaple atom.

CSS

With CSS or "Cascading Style Sheets" you can add style to the structure of your web page effecting how it is viewed by the user.

You can have several CSS files or sheets and you can reference them in your html. "Cascading" means that rules are applied not only to the elements they directly match, but also to all of those elements' child elements. However, if a child element has multiple, overlapping rules defined for it, the more specific rule takes effect.
